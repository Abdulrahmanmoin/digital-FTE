Maximize productivity with these shortcuts 
Become a Claude Code power user. Master these shortcuts to maximize your 
coding productivity gains with the tool.

Top shortcuts:
`tab` - Enable extended thinking before sending
`#` - Add a memory to Claude Code
`shift+tab` - Toggle between Plan Mode and auto-accept mode 




Get more shortcuts  
<https://docs.claude.com/en/docs/claude-code/interactive-mode> 






Run Claude Code in your IDE 
Run `claude` in the integrated terminal of VS Code 
<https://docs.claude.com/en/docs/claude-code/vs-code> or JetBrains IDEs 
<https://docs.claude.com/en/docs/claude-code/jetbrains> to unlock extra 
capabilities:


 * Changes appear directly in your editor

 * Claude Code can see diagnostic errors and fix them

 * Claude Code sees your open files and highlighted text for added context 






Quick context with @ mentions 
Need to add a file to the conversation? Type `@` and start typing the filename:

> @auth.ts @user.model.ts explain how authentication works across these files 






Pull the latest documentation 
Claude Code can search the web and fetch documentation in real-time:

> What's the latest way to handle authentication in Next.js 16? Check the 
official docs.

Pro tip: When you finish a feature, clear the conversation with `/clear` so 
Claude Code stays focused on what matters
